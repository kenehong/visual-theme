Kimbap — Style Comparison

Same subject, 7 CSS illustration approaches

A. Kawaii Flat

LINE Friends / Sticker Art

Pure flat colors, thick outlines, bubbly shapes, cute face. Saturated and friendly.

B. Soft Illustration

Notion / Linear Style

Flat base with subtle inner shadows. No outlines, matte feel. Slightly muted colors.

C. Retro Sticker

Vintage Label / Screen Print

Bold outlines, halftone dot overlay, hard offset shadow. Warm vintage palette.

D. Watercolor

Children's Book / Gentle Wash

Soft edges, color bleeds, gradient washes on paper-like background. Pastel palette.

E. Geometric Minimal

Bauhaus / Swiss Design

Simplified rectangles, max 5 colors, no shadows or gradients. Clean and mathematical.

F. Pixel Art

8-bit / Retro Game

12px grid, box-shadow pixel drawing. Seaweed ring, rice, fillings in chunky pixels. Game-ready.

G. Bold Pixel

Retro Illustration / Game Art

Thick outlines meet stepped pixel edges. Clean shapes on a chunky grid. Best of both worlds.
