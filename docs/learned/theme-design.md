# Theme Design

## 품질 기준
- 테마는 `featured: false`로 시작 → 유저 프리뷰 승인 후 `true`
- 베이스 프리셋(bold, clean, dense, playful)은 갤러리에 넣지 않음

## 핵심 패턴
- 실사 필요 주제(음식, 도자기 등)는 CSS만으로 부족 → 데모에 이미지 옵션 고려
- 비슷한 컨셉 중복 시 퀄리티 높은 하나만 keep (korean-neon vs radio-city)
- 원본 레퍼런스의 킥 컬러를 반드시 primary/accent로 살릴 것
- visual-heavy 테마(glass, neon)는 트랜지션/프레임에 더 투자 필요
- 데모 = 컴포넌트 나열이 아님 → 실제 사용 시나리오가 보여야 함
